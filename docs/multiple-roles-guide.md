## ðŸ’¡ How Permissions Work with Multiple Roles

### **Rule: Combined Permissions (Union)**

When a user has multiple roles, they get **ALL permissions** from **ALL roles**:

```
Faculty Role Permissions:
- View Students
- Manage Grades
- View Subjects

Registrar Role Permissions:
- Manage Enrollments
- Generate Reports
- View Academic Records

User with BOTH roles gets:
âœ… View Students
âœ… Manage Grades
âœ… View Subjects
âœ… Manage Enrollments
âœ… Generate Reports
âœ… View Academic Records
```

**It's an OR operation, not AND!**
